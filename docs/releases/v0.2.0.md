# EventSubtitles v0.2.0

## Download

```text

```

It contains `EventSubtitles.app` for Apple Silicon Macs.

SHA-256:


## Highlights

- Reworked operator UI into persistent left controls plus task-focused workspaces.
- Added Live, Style, Glossary, Logs, Models, Translation, Audio, and Output workspaces.
- Improved Glossary, Models, and Output layouts for narrower windows.
- Renamed the capture picker to use task-focused labels:
  - Demo captions.
  - Live subtitles (WhisperKit).
  - Record audio only.
- Added generated macOS app icon support to the bundle build.
- Kept fully local session logging with transcripts, SRT files, JSONL segments, and audio recording.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
