# EventSubtitles v0.2.2

## Download

```text

```

It contains `EventSubtitles.app` for Apple Silicon Macs. The app is displayed as `Subtitles`.

```text

```

## Highlights

- Added calm caption display controls for stable public output: Calm Blocks, Live Roll-up, and Fast Draft.
- Split draft transcript text from public caption text so raw ASR partials do not constantly rewrite the HDMI output.
- Added display-flow settings for stability, commit delay, minimum hold, maximum latency, and hidden unstable words.
- Reworked the Glossary workspace into a row-based term editor with add/edit/delete controls.
- Added glossary alias groups, quality checks, session suggestions, advanced bulk edit, and JSON/CSV import/export.
- Improved Style layout so core visual controls stay usable while preview and display-flow controls remain visible.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
