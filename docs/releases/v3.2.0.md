# EventSubtitles v3.2.0

## Download

The GitHub release asset is:

```text
EventSubtitles-v3.2.0-macos-arm64.zip
```

It contains `EventSubtitles.app` for Apple Silicon Macs. The app is displayed as `Subtitles`.

SHA-256:

```text
c0386bedcee1c13b25a322fbd2588f062825105ccabd67f6ba793c0512b2151b
```

## Highlights

- Added a single live audio capture pipeline shared by the meter, CAF recorder, and WhisperKit.
- Added TV-style live roll-up captions with line hold, idle flush, and auto-clear controls.
- Added panic blank and output blank controls for live-event recovery.
- Improved output-window lockdown for external display use.
- Added word-confidence aware caption stability and more rolling-caption tests.
- Fixed review issues around toolbar panic behavior, Stop-time rolling flush, output-width reporting, queued CAF writes, and audio restart recording continuity.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesCoreUnitTests
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
