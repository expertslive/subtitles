# EventSubtitles v0.2.1

## Download

The GitHub release asset is:

```text
EventSubtitles-v0.2.1-macos-arm64.zip
```

It contains `EventSubtitles.app` for Apple Silicon Macs. The app is displayed as `Subtitles`.

SHA-256:

```text
b541b969a95ec0b1c0ad52aed3b1b758a0b2f0ffd71911004fafdd21c7542949
```

## Highlights

- Added a custom app icon from `Assets/AppIconSource.jpg`.
- Updated the visible app/window/menu name to `Subtitles`.
- Added a short app-specific About panel.
- Added Glossary JSON/CSV import and export.
- Added fine X/Y caption positioning and smaller font sizes.
- Added a full-height Live history column.
- Added Models guidance for offline preparation, first-start warmup, memory usage, and Activity Monitor checks.
- Added automatic sleep prevention while a subtitle session is running.
- Kept WhisperKit loaded after Stop so later Starts resume faster.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
