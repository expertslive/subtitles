# EventSubtitles v3.0.0

## Download

The GitHub release asset is:

```text

```

It contains `EventSubtitles.app` for Apple Silicon Macs. The app is displayed as `Subtitles`.

```text

```

## Highlights

- Redesigned the operator window with a `NavigationSplitView` sidebar, global toolbar, Now panel, and dedicated workspace detail area.
- Split the large operator view into focused Live, Style, Glossary, Logs, Models, Translation, Audio, and Output workspace files.
- Reworked Style as a preview-first workspace with independent control columns so the most-used controls stay visible.
- Stabilized toolbar sizing so hover/focus does not move session, capture, language, mode, meter, or output controls.
- Improved Calm Blocks so the last spoken sentence is published after the maximum latency even when WhisperKit does not emit a final segment immediately.
- Added smoke coverage for Calm Blocks idle-tail flush behavior.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
