## Download

```text

```

It contains `EventSubtitles.app` for Apple Silicon Macs. The app is displayed as `Subtitles`.

SHA-256:

```text
834c4d7a294beabeb733e0320c3f8f3814887f295987376548cd0a4f48964a82
```

## Highlights

- Fixed panic blank so it no longer queues a subtitle backlog. While the output is blanked, incoming transcripts no longer feed the live caption pipeline, so nothing accumulates behind the hidden surface. Finals are still recorded to transcript history and the session log, and unblanking resets the pipeline so captions resume clean from live audio instead of replaying what was spoken during the blank.

## Validation

Release validation commands:

```bash
swift build --product EventSubtitles
swift run EventSubtitlesCoreUnitTests
swift run EventSubtitlesSmokeTests
./scripts/build_app_bundle.sh
codesign --verify --deep --strict build/EventSubtitles.app
```

## First Launch

The app bundle is ad-hoc signed for local testing. macOS may require opening it from Finder with Control-click > Open the first time.
